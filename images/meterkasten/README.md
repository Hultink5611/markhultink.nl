# Meterkast portfolio foto's

Plaats hier 4 foto's van afgeronde meterkasten als:

- `01.jpg`
- `02.jpg`
- `03.jpg`
- `04.jpg`

De thumbs in het portfolio-paneel zijn vierkant en gebruiken `object-fit: cover`, dus afbeeldingen worden automatisch gecropt tot een mooi vierkant — afmeting / aspect ratio van het origineel maakt niet uit. Voor snelle laadtijd: exporteer ze op max ~600×600px en ~80% JPEG-kwaliteit.

Ontbrekende bestanden worden automatisch afgevangen (onerror → img verwijderd) waardoor het gearceerde placeholder-patroon zichtbaar blijft.
